# 文风规范库

> 目标：平台输出的所有文字（策略说明、分析文章、复盘帖、推文）听起来像一个懂球的人在说话，而不是 AI 在生成报告。
> 本文件持续更新——每次改出好案例，存进「改写案例」区。

---

## 一、核心原则

### 1. 结论前置，不绕弯
先给判断，再解释。读者扫一眼第一句就知道你在说什么。

### 2. 有态度，不模糊
选一个立场说清楚。"有一定概率"这种话谁都会说，没有价值。

### 3. 数字说话
赔率、概率、档差、射门数——能用数字的地方不用形容词。

### 4. 承认不确定要干脆
说不好就说说不好，不要用"仍需观察"来掩盖。

### 5. 短句，克制分段
一段 2-4 句。不写长篇分析，写能发出去的内容。

## 二、禁忌词 → 替换方式

| 禁忌 | 替换 |
|------|------|
| 值得注意的是 | 删掉，或直接说那件事 |
| 不得不说 | 删掉 |
| 综上所述 | 删掉，直接写结论 |
| 整体来看 / 总体来说 | 删掉 |
| 需要指出的是 | 删掉 |
| 从某种程度上来说 | 删掉 |
| 可以预见 | 删掉 |
| 与此同时 | ，（逗号）或 而且 |
| 此外 | 删掉，或 另外（少用）|
| 在一定程度上 | 有点 / 略微 |
| 不可否认 | 删掉 |
| 不失为 | 改成直白表达 |
| 显而易见 | 删掉，直接说结论 |
| 毋庸置疑 | 删掉 |
| 有目共睹 | 删掉 |
| 值得关注 | 删掉，或说具体什么值得关注 |
| 较为 / 相对较 | 直接给数字 |
| 一定的 | 删掉 |

---

## 三、结构禁忌

- ❌ 先铺背景再给结论（读者等不及）
- ❌ 每段都用「首先…其次…最后…」
- ❌ 结尾加「总的来说 / 综上」复述一遍已经说过的话
- ❌ 「这场比赛双方实力相当，存在多种可能性」——废话，所有比赛都这样
- ❌ 长破折号（——）连续出现两次以上，读起来像模板

## 四、改写案例

> 每次改出好对比，存在这里。格式：原文 → 改后 → 改了什么。

---

### 案例 001 · 策略说明（2026-06-12）

**原文（AI生成）**
> 综合考虑多方面因素，法国队在本次小组赛中有较大可能获得第一名，值得关注。

**改后**
> 法国这场稳，不出意外拿小组第一，赔率 1.12x 打的是确定性不是回报。

**改了什么**
- 删"综合考虑多方面因素"全句废话
- "较大可能"→直接下判断
- 加赔率数字，让读者知道值不值
- "值得关注"→删掉，这句话什么信息都没给

---

### 案例 002 · 策略说明（2026-06-12）

**原文（AI生成）**
> 摩洛哥末轮比赛存在较大不确定性，双方出线形势均有变数，需要进一步观察才能确定最优押注方向。

**改后**
> 摩洛哥末轮说不好——平局出线、赢也出线，两队心思不统一，这就是摇摆盘。Polymarket 赛中随时可以买，等上半场结束再看更稳。

**改了什么**
- "存在较大不确定性"→"说不好"，干脆
- "需要进一步观察"→直接给操作建议（等半场）
- 加 Polymarket 实盘特性，说明怎么处理不确定性

---

### 案例 003 · 比赛中观测策略描述（2026-06-12）

**原文（旧版）**
> 据半场数据（比分/控球/射门/射正）输出对应盘口。半场0-0但控球压制明显→押下半场进球；角球密集→押角球盘。不抢市场反应，依据已发生的场上事实。

**改后**
> Polymarket 比赛全程开盘，随时可以买卖。半场结束后，根据实际数据（控球率、射门/射正数、半场比分）再入场，比赛前买信息更充分。典型场景：半场0-0但主队控球65%射门10+，此时押「全场大分」或「主队胜」赔率还在，值得进。

**改了什么**
- 原版说"不抢市场反应"像是在解释原则，改版直接说用法
- 加了具体数字（65%/10+）让场景可感知
- 点出 Polymarket 的核心优势：赛中开盘

## 五、推文文风（单独一区）

推文比分析文章更短、更口语、更有个性。

**原则**
- 140字内能说完就别写更长
- 开头不用"今天"——那是废话
- 带具体盘口和赔率，不然读者不知道怎么用
- 结尾不用"欢迎交流"之类客套话

**推文模板（不是固定格式，是参考节奏）**
```
[比赛] 法国 vs 澳大利亚，今晚21:00
[判断] 押大分，Over 2.5，赔率1.82x
[理由] 法国S级打D级，进攻动力足，澳大利亚守不住
[提示] 赔率还行，不是捡钱但逻辑清楚
```

**反例（不要这样写）**
> 今天为大家带来一场值得关注的比赛分析。法国队综合实力较强，从多方面来看，大分是较为合理的选择，感兴趣的朋友可以参考。

## 六、复盘文风

复盘的核心是**真实**，不挑好看的说。

- 命中了：说中了什么，赔率值不值
- 没中：说哪里判断错了，下次怎么改
- 不要把运气说成实力，也不要把实力说成运气
