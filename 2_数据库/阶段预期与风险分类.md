# 阶段预期与风险分类（每队地板/天花板）

> 用途：为 [三榜规则与展示规范](../3_策略分析库/三榜规则与展示规范.md) 提供"阶段"维度的可执行数据底座。
> 风险 = 队伍实力 × 分组 × **阶段预期**。本表定义每队的地板/天花板。
> 定稿：2026-06-11 | ⚠️ 赛前静态推演，小组赛开打后随积分动态修正（P0.1 不臆造）

## 一、赛制与阶段（2026 新制）

48队 → **32队出线**（12组各前2 + 8个最佳第三）→ 32强 → 16强 → 8强 → 4强 → 决赛。
**只有16队小组出局。**

| 缩写 | 阶段 |
|------|------|
| GS | 小组出局 |
| R32 | 32强 |
| R16 | 16强 |
| QF | 8强 |
| SF | 4强 |
| F | 决赛 |
| W | 冠军 |

> ⚠️ **关键（注意别混淆"风险"与"区分度"）**：32/48 队出线，"进32强"对强队近乎白送。
> - 作为**下注风险**：它就是**低风险**（近乎必然，押输概率极低），照常进低风险榜。
> - 但作为**设定地板的锚点**：R32 **没有区分度**（人人都过，对法国/韩国/南非都成立），
>   所以把强队的"地板"上移到 **R16**——那才是既高把握、又能区分实力的阶段。
> - 地板设在 R16 **不会**把"进32强"踢出低风险：R32 比 R16 更浅，按"主张≤地板→低风险"，
>   强队进32强仍是低风险，且是最稳一档。它的短板是**回报**（盈亏比≈1.0x），不是风险。
>   按"不按价格砍盘"原则，照常展示并标盈亏比，由用户判断值不值。
> - 中下游队的地板才落在 GS/R32。

---

## 二、地板 / 天花板 的定义

- **地板**：高把握**至少**能到的阶段。盘口主张 **≤ 地板 → 低风险**。
- **天花板**：现实最好情况能到的阶段。盘口主张 **≥ 天花板 → 高风险·长赔**。
- **地板与天花板之间**：该队的现实区间，主张落此 → **高风险·摇摆**（结果无把握）。

## 四、风险分类映射（喂给三榜）

### 锦标赛进程类盘口（夺冠/进X强/出线）
对"队X 到达阶段Y"的盘：
- **Y ≤ 该队地板 → 低风险**。例：法国进R16(地板R16，≤)、葡萄牙出线。
- **地板 < Y < 天花板 → 高风险·摇摆**。例：日本进QF(地板R32/顶QF区间内)。
- **Y ≥ 该队天花板 → 高风险·长赔**。例：摩洛哥夺冠(顶SF)、墨西哥进SF(顶QF)。

### 单场对阵类盘口（胜负/让球/总分）
- **实力差≥2档**（如S打D、A打D）→ 强队胜 = **低风险**；强队让球大分 = 低风险偏稳。
- **同档或差1档且都中游**（B vs B、A vs A）→ **高风险·摇摆**。
- **弱队掀翻强队 / 鱼腩爆冷** → **高风险·长赔**。

### 特别角度：强队"爆冷出局"的反面盘（高风险·长赔）
低风险的反面,往往是高风险榜里值得小仓博的标的——**同一枚硬币的两面**。

**逻辑**：强队进32强=低风险·低回报(≈1.0x);反过来,"某支S级强队**无法**进32强 / 小组出局"这种盘,
平时隐含概率极低(约2–5%)、**盈亏比极高(20–50x)**,但**真的会发生**:
- 德国 2018、2022 **连续两届均在小组赛出局**(2018 F组垫底、2022 E组第3)
  - 其中 2018 德国是**卫冕冠军**(2014夺冠)→ 卫冕冠军魔咒;2022 已非卫冕(2018冠军为法国),纯属强队翻车
- **卫冕冠军魔咒**(夺冠次届即小组出局):法国2002、意大利2010、西班牙2014、德国2018——四届连续应验
- 其它夺冠热门折戟小组赛亦不鲜见

**为什么归入高风险·长赔**：强队出局隐含概率低(约2–5%)、盈亏比高(20–50x),历史上确有发生。
作为"低风险·进32强"的反面盘,自然进**高风险·长赔**子区**呈现并标盈亏比**,由用户自行决定是否小仓表达观点。

**纪律(重要)**：
- **不追信息差**——伤停、内讧、"盘口未反应"等抢跑式机会一律不做。预测市场反应很快,这类边际我们不参与。
- 平台只做**分类与呈现**(按风险归档 + 标盈亏比),不声称"比市场更早知道"。
- 分类随**真实赛果**动态更新(强队首轮爆冷→天花板/地板下调),不在赛前臆造。

### 与盈亏比的关系（独立轴）
风险定档后，**每条仍必标盈亏比**：
- 低风险榜：盈亏比通常低（如1.1–1.5x），看的是稳。
- 高风险·长赔：盈亏比高（5x+），小仓博。
- 高风险·摇摆：盈亏比中等（2–3x），博惊喜。
