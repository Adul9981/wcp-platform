## 字段结构（teams.json）
```
teams[<英文名>]:
  静态(权威): name_en, name_zh, group, confederation, fifa_rank, tier, is_host
  动态(随赛程更新):
    played,w,d,l,gf,ga,pts   # 小组积分
    group_rank               # 组内排名
    status                   # 未开赛/小组赛中/已出线/已淘汰/已晋级X强
    recent_form[]            # 近期战绩
    key_absences[]           # 伤病/停赛缺阵（策略：阵容情况）
    fatigue_note             # 疲劳/赛程密度备注
    last_updated
```

## 更新机制（活数据，随世界杯进程变化）
