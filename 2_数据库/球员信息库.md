## 范围与原则
- **只收** Polymarket 球员盘点名的 16 名球星（不贪多）
- **静态字段**(所属国家队/位置/是否主罚点球·定位球·是否进攻核心)=稳定公开事实，存疑保守标 False
- **动态字段**(能否参赛/伤病)赛前留空，随消息更新 → 对应"反向爆冷"机会

## H2H 进球期望代理评分（players.py）
诚实：这是**启发式代理，非精确预测**。
```
得分 = 球队层次(S5/A4/B3/C2/D1，代理出场场次) + 角色(核心+2 / 点球手+1.5 / 定位球+0.5)
```
两球员对决取得分高者；差距<1.0不推；**我方所选若市场价<40%(市场强烈反对)→不逆势推荐**。

### 已生效方向示例
- 亚马尔 vs 姆巴佩 → 押姆巴佩(法国S级·点球手·核心)
- 巴尔韦德 vs B费 → 押B费(葡萄牙S级·点球手·核心)
- 内马尔 vs 维尼修斯(同队) → **不推**(市场仅给内马尔19%，代理逆市场被抑制)
