📋 文风指南（提交前参考，去 AI 味）

核心原则

结论前置 · 数字说话 · 有态度不模糊 · 短句（一段 2-4 句） · 承认不确定性要干脆

禁忌词（提交前删掉）

值得注意的是 / 不得不说 / 综上所述 / 整体来看 / 需要指出的是 / 从某种程度上 / 可以预见 / 与此同时 / 一定程度上

对比示例

✅ 法国这场稳，赔率 1.12x 打的是确定性不是回报。小组第一没悬念。
❌ 综合考虑多方面因素，法国队整体来看夺得小组头名的可能性较大。
✅ 摩洛哥末轮这局说不好——平局也出线、赢也出线，双方动力不统一，摇摆。
❌ 值得注意的是，摩洛哥末轮存在较大不确定性，需进一步观察。
