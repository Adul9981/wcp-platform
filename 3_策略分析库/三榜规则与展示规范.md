# 三榜规则与展示规范

> 平台核心展示逻辑。定稿日期：2026-06-11（与用户逐条商定）
> 本规范取代旧的"最值得关注/最被忽视/最大盈亏比"三榜框架（见文末迁移说明）

---

## 〇、最高原则：宁可空，不可乱

用户明确（2026-06-11）：**"展示的选题可以少，但不能乱展示"**。落成三条硬机制：

1. **只靠质量门槛过滤，不靠数量上限**。不设每日条数上限——因为一场比赛本就有胜负/让球/总分/角球等几十个盘，数量不是问题。
2. **不按价格砍任何区间**。不存在"中间地带不展示"。掷硬币区也可能有惊喜，全部保留。
3. **每个选题必须能说清"为什么在这一榜"**——有队伍实力/分组/阶段的明确依据。说不清依据的，不进任何榜。宁可某榜空着 + 写明原因，绝不为填满而放低标准。

---

## 一、风险的定义（核心：队伍维度，不是价格维度）

风险由三样决定：**队伍实力 × 分组情况 × 可能走到的阶段**。
价格（隐含概率）**不**用来定义风险——它只用来算盈亏比/判断值不值，是独立的另一根轴。

### 可操作判据
> **风险 = "盘口的主张" 与 "队伍实力/分组现实能到达的位置" 之间的差距。**

| 情形 | 风险 | 例子 |
|------|------|------|
| 主张 ≤ 队伍"地板"（稳的） | **低风险** | S级强队打D级鱼腩；A级队从弱组出线；强队进淘汰赛 |
| 主张 ≈ 队伍"天花板"或更高 | **高风险·长赔** | 中游队夺冠；鱼腩爆冷掀翻强队 |
| 两队势均力敌、结果无把握 | **高风险·摇摆** | 两个B级队对踢；死亡之组内战 |

同一支队、不同主张，风险不同：
- 法国"进淘汰赛" → 低风险
- 法国"夺冠" → 高风险·长赔

数据依据来自 [国家队信息库](../2_数据库/国家队信息库.md)（队伍分级 + 分组 + 阶段预期）。

## 二、三榜定义

### 1️⃣ 今日关注（时间轴 · 回答"今天该看什么"）
与风险正交——榜内选题本身可能是低风险也可能是高风险，各自标注。

**结构**：
- **顶部（主区）**：今天开赛 / 今天结算的比赛。
- **下部（副区）**：今天有催化剂或里程碑的事件，例如：
  - 哪些国家已锁定 / 即将锁定进入淘汰赛
  - 谁的出线悬念最大（末轮生死）
  - 新上线且值得关注的盘、价格大幅移动的盘、关键伤停消息

**空榜规则**：赛事未开打的日子，主区可为空，副区仍可有内容；都没有则显示空 + 原因。

---

### 2️⃣ 低风险事件（高确定性 · 稳）
**准入门槛（全部满足才进）**：
- [ ] 主张 ≤ 队伍实力地板（按上方判据）
- [ ] 有明确的实力/分组/阶段依据可解释
- [ ] 有有效报价（能算盈亏比）

**典型内容**：S级打D级的胜/让胜/大分；强队进淘汰赛；强队小组头名（弱组）。
**注意**：低风险 ≠ 低赔就值得。仍要标盈亏比，让用户自己判断值不值。

---

### 3️⃣ 高风险事件（低把握 · 博，含两个子区）

#### 子区 A：长赔（纯低概率高赔）
- 主张 ≥ 队伍天花板：中游/弱队夺冠、鱼腩爆冷、冷门晋级。
- 特征：隐含概率低、盈亏比高，小仓博弈。

#### 子区 B：摇摆 ⭐（用户要求单开）
- 势均力敌、结果无把握、可能有惊喜的对决。
- 两个B级队对踢、死亡之组内战、实力接近的淘汰赛。
- 与子区A分开：摇摆是"谁都可能赢"，长赔是"基本不可能但万一"。

**准入门槛**：
- [ ] 有逻辑说得清为什么可能命中 / 为什么摇摆（不是随便一个便宜盘）
- [ ] 有有效报价

## 三、每个选题的必含字段

无论在哪一榜，每条选题都必须展示：

| 字段 | 说明 |
|------|------|
| 中文标题 | glossary 翻译，保留英文原文可追溯 |
| 盘口/方向 | 胜负 / 让球 / 总分 / 角球 / 出线 / 夺冠 等 |
| **盈亏比（回报率）** | **必含**。= 1/隐含概率，赢1注的毛回报倍数。用户明确要求每个下注都标清 |
| 隐含概率 | 当前价格 |
| 风险归属 | 低风险 / 高风险·长赔 / 高风险·摇摆 |
| 入榜理由 | 一句话说清实力/分组/阶段依据（可解释，不黑箱） |
| 涉及队伍 | 关联国家队信息库 |
| 成交量 / 流动性 | 参考，**不作为剔除依据**（低流动是先发机会的正常状态） |
| 是否新上线 | 先发机会标记 |
| 开赛时间 / 链接 | |

## 四、与旧三榜的迁移说明

| 旧榜（按发现角度） | 去向 |
|------------------|------|
| 🎯 最值得关注 | 拆解 → 今日关注（时间） + 低风险（确定性强的部分） |
| 💎 最被忽视 | 不再独立成榜 → "新上线/低关注"降为**字段标记**，可出现在任意榜 |
| 📈 最大盈亏比 | 并入 **高风险·长赔** 子区 |

代码层（`platform/wcp/boards.py` + `scoring.py`）需按本规范重构：
- 评分轴从"价格/流动性/成交"改为"队伍实力×分组×阶段"的风险分类 + 盈亏比
- 风险分类依赖 [国家队信息库](../2_数据库/国家队信息库.md) 的分级与阶段预期
- 保留可解释 reason、保留"低流动不过滤"、保留中文翻译

> ⚠️ 本文件只定规则。代码重构在用户确认后另行进行。
