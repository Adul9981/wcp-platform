# 策略触发检查清单（赛前快速核查）

> 每场比赛赛前填写，5分钟内完成。
> 对齐 [核心策略清单](核心策略清单.md)：仅 A/B/C/D/F/G/I 七条启用。
> 原则：**不追信息差、不写仓位**（仓位用户自定），每条只看模式+方向+盈亏比。

---

## 基本信息

```
场次：__________ vs __________
小组：___  轮次：第_轮
时间：___________
双方积分：队A ___ 分 / 队B ___ 分
晋级情况：队A [已出 / 摇摆 / 已晋]  队B [已出 / 摇摆 / 已晋]
队A实力分级：[S/A/B/C/D]  队B实力分级：[S/A/B/C/D]
```

---

## 赛前策略触发检查

### ✅ 策略A（强打弱大分）
- [ ] 实力差 ≥ 2档？
- [ ] 强队有晋级/净胜球动力？
- [ ] 弱队无守平动力？
→ **触发 → 押大分 Over；可选准确比分（接F）**

### ✅ 策略B（强队确定性）
- [ ] S/A级队，主张 ≤ 其"地板"阶段（见[阶段预期与风险分类](../2_数据库/阶段预期与风险分类.md)）？
- [ ] 确认非主力轮换？
→ **触发 → 低风险榜；标盈亏比（通常低，由用户判断值不值）**

### ✅ 策略C（摇摆末场生死战）
- [ ] 第三轮比赛？
- [ ] 确认两队同时需要赢（或平局悖论）？
- [ ] 另一组同时踢（确认没有意外出线）？
→ **触发（赛前不预设，按真实积分）→ 高风险·摇摆，视场景选大分/小分/特定队**

### ✅ 策略F（准确比分·小仓博高赔）
- [ ] S级打D级、火力充足？
→ **触发 → 高风险·长赔，列 3-0/4-0/3-1 等比分及盈亏比**

### ✅ 策略G（东道主加成）
- [ ] 美国/墨西哥/加拿大参赛、在本国城市踢？
→ **触发 → 低风险偏稳；东道主主胜/让球（仅作分类呈现，不评估"市场是否定价错"）**

### ✅ 策略I（保守踢法小分）
- [ ] 一队已锁定出线无净胜球压力 / 另一队已出局？
→ **触发（需赛中确认状态，赛前不预设）→ 押小分 Under / 平局**

## 比赛中观测（策略D · 非赛前预设）

> D 是**比赛中**依据可观测数据输出对应盘，不抢市场反应速度。赛前此栏留空，开赛后观测：

| 观测维度 | 关注点 | 可输出盘口 |
|---------|--------|-----------|
| 半场比分 | 半场进球 / 半场不进球 | 半场大/小分、下半场进球与否 |
| 控球率 | 一方明显压制 | 全场大/小分中段研判 |
| 射门/射正 | 射门多但未进 | 角球数盘、后续进球盘 |
