# 策略触发检查清单（赛前快速核查）

> 每场比赛赛前填写，5分钟内完成

---

## 基本信息

```
场次：__________ vs __________
小组：___  轮次：第_轮
时间：___________
双方积分：队A ___ 分 / 队B ___ 分
晋级情况：队A [已出 / 摇摆 / 已晋]  队B [已出 / 摇摆 / 已晋]
队A实力分级：[S/A/B/C/D]  队B实力分级：[S/A/B/C/D]
```

---

## 策略触发检查

### ✅ 策略A（强打弱大分）
- [ ] 实力差 ≥ 2档？
- [ ] 强队有晋级/净胜球动力？
- [ ] 弱队无守平动力？
- [ ] 大分线 ≤ 3.5？
→ **全部✅ → 触发，押大分 Over**

### ✅ 策略B（强队确定性）
- [ ] S/A级队赢了前两场？
- [ ] 晋级赔率 > 1.08？
- [ ] 确认非主力轮换？
→ **全部✅ → 触发，押主胜或亚盘**

### ✅ 策略C（摇摆末场生死战）
- [ ] 第三轮比赛？
- [ ] 确认两队同时需要赢（或平局悖论）？
- [ ] 另一组同时踢（确认没有意外出线）？
→ **全部✅ → 触发，视场景选大分/小分/特定队**

### ✅ 策略I（保守踢法小分）
- [ ] 一队已锁定出线且无净胜球压力？
- [ ] 另一队已出局？
- [ ] 历史该队保守踢法记录？
→ **全部✅ → 触发，押小分 Under**

### ✅ 策略G（东道主加成）
- [ ] 美国/墨西哥/加拿大参赛？
- [ ] 在本国城市踢？
- [ ] 赔率是否反映了主场加成？（若已反映则跳过）
→ **全部✅ → 触发，加 0.25 档主场溢价**

### ✅ 策略H（疲劳差套利）
- [ ] 两队休息日差 ≥ 2天？
- [ ] 体能消耗大的队是哪队？
→ **触发 → 对体能充足队额外加注**

## 盘中事件预设

**如果发生以下事件，执行对应操作：**

| 事件 | 时间窗口 | 预设操作 |
|------|---------|---------|
| 红牌 | 前60分钟 | 观察赔率移动，若过激则反向 |
| 0-0 到 第60分钟 | — | 押 Over 1.5（若双方有进攻动力）|
| 补时 ≥ 5分钟公布 | — | 押补时进球（如有此盘） |
| 比分落后方扑点 | — | 短期内押比分不变（窗口1-2分钟）|
